    • Instalación y configuración de entorno

    Step 1.

    Una vez tengamos VirtualBox instalado y el ISO de WindowsServer listo, lo ejecutaremos y lo primero que deberemos hacer es crear una nueva partición del tamaño máximo.

    Step 2.

    Posteriormente, se formatea el disco de 49.5 GB y con esto empezará a preparar todo para la instalación.

    Step 3.

    Se reiniciará y empezará a preparar el equipo.

    Step 4.

    Se configura una contraseña, considerar que debe contener números, letras, símbolos.

    Step 5.

    Para poder ingresar se debe usar la combinación de teclas: ctrl+supr. Una vez la instalación haya quedado correctamente, se nos abrirá el “Administrador del servidor” donde veremos todas las opciones y características d en nuestro equipo. En el menú izquierdo, nos iremos a “Panel local” y ahí configuraremos el nombre del equipo y la Dirección IP.

    Step 6.

    Por default aparece un nombre alfanumérico, nosotros solo damos click para modificarlo. Nosotros le hemos puesto “windows-server-2019”. Una vez hayamos modificado el nombre daremos click en aceptar y finalmente se reiniciará el equipo.

    Step 7.

    Después de haber configurado el nombre del servidor, nos iremos a modificar la Dirección IP; seguiremos la opción donde dice Ethernet > Propiedades > Propiedades: Protocolo de Internet versión 4 (TPC/IPv4) y ahí será donde configuraremos la Dirección IP, en nuestro caso le hemos puesto “192.168.100.2”. Posteriormente, la Máscara de subred es 255.255.255.0 y finalmente, para el Servidor DNS preferido colocaremos la misma Dirección IP.

    Nota: Consideremos que tiene que ser una IP diferente a la de nuestro equipo de Windows local (162.198.100.20).

    Step 8.

    En el menú superior, daremos click en Administrar > Agregar roles y características > Roles de servidor y aquí nos aparecerá una lista de opciones, nosotros seleccionaremos “Servicios de Dominio de Active Directory”.

    Step 9.

    Damos click en “Siguiente”, nuevamente nos mostrará las características, damos en “Siguiente” hasta que nos aparezca la opción de “Instalar”. Esperaremos a que la instalación concluya, esto nos permitirá trabajar con los dominios.

    Step 10.

    Una vez la instalación haya quedado finalizada daremos click en “Cerrar”.

    Step 11.

    En el menú superior, la banderita nos va a indicar que tenemos un detalle, y es para poder configurar el dominio

    Step 12.

    Daremos click donde dice “Promover este servidor a controlador de dominio” y se nos abrirá una ventana y daremos click en la opción de “Agregar un nuevo dominio”; nosotros hemos puesto “equipo6.net” y damos click en “Siguiente”.

    Step 13.

    Nos pedirá seleccionar el Nivel funcional, dejamos el que viene por default, colocaremos una contraseña nueva y daremos click en “Siguiente”. (Considerar los mismos requisitos de seguridad para la contraseña ).

    Step 14.

    Nos aparecerá el nombre del dominio que hemos colocado anteriormente, damos click en “Siguiente” y si todo se ha configurado correctamente deberemos llegar a la pestaña de “Instalar”.

    Step 15.

    Una vez que la instalación haya quedado finalizada, nos pedirá reiniciar el equipo, lo hacemos y con esto debe haber quedado esta configuración para el dominio. (El reinicio puede tardar un poco más de lo esperado).

    Instalación de MYSQL Server

    Step 1.

    Ejecutamos el archivo llamado Setup.exe (proporcionado por el profesor), realizaremos una instalación típica, por lo que le daremos siguiente hasta llegar al apartado de inicio de sesión.

    Step 2.

    Seleccionaremos el salto de inicio de sesión y le continuamos.

    Step 3.

    Es de gran importancia que esta casilla esté seleccionada puesto que configuraremos mysql de inmediato; una vez seleccionada, procedemos a terminar la instalación.

    Step 4.

    Se abrira una nueva carpeta, nosotros le daremos next hasta llegar a esta ventana, en la cual seleccionaremos la opcion de standard configuration, le damos siguiente.

    Step 5.

    En la siguiente ventana es muy importante seleccionar ambas opciones marcadas, pues la primera se encarga de instalar mysql como un servicio y la segunda incluye mysql en el bin directory, continuamos.

    Step 6.

    En la siguiente pantalla nosotros debemos crear una contraseña para el usuario root, en este caso y con el objetivo de la práctica, la contraseña será “123”, al igual que habilitamos el acceso a máquinas remotas. las prácticas anteriores no suelen ser buenas pues debilitan el sistema pero es importante para la práctica que se está llevando a cabo. continuamos.

    Step 7.

    En esta pantalla nosotros haremos click al botón Execute, es importante que todas las opciones de arriba queden palomeadas puesto que de no ser asi, tendríamos que borrar todas las carpetas llamadas mysql y reinstalar el servicio.

    Step 8.

    Con esto finalizaríamos la instalación de MYSQL Server.

    MYSQL Server

    Step 1.

    Para entrar a mysql desde powershell, es necesario ingresar el comando subrayado, para luego ingresar la contraseña del usuario root ue previamente configuramos, una vez ingresado todo, estamos dentro de mysql.

    Step 2.

    Creamos una nueva base de datos llamada datos con el comando mostrado, siempre cuidando el uso de punto y coma después de cada instrucción.

    Step 3.

    Para comprobar la creación de la base de datos se usa el comando resaltado, y como se puede apreciar, se creó correctamente.

    Step 4.

    Para ingresar a alguna base de datos se aplica el comando subrayado, y como muestra el output, la acción se realizó correctamente.

    Step 5.

    Creamos una nueva tabla dentro de la base de datos, la cual es llamada integrantes, posee tres columnas: nombre, apellido y matricula, es creada con el siguiente comando: create table integrantes(Nombre varchar(20), Apellido varchar(20), Matricula varchar(10));

    Step 6.

    Usamos el comando show tables; para ver la tabla creada, llamada integrantes.

    Step 7.

    Ingresamos el comando insert into integrantes values(); para poder ingresar datos a nuestra tabla.

    Step 8.

    Se emplea el comando select * from integrantes; para poder ver el contenido de nuestra tabla.

    Step 3.

    Step 4.

    Configuraremos el TCP, agregando el puerto correspondiente a mysql (3306), siguiente.

    Step 5.

    Permitimos todas las conexiones,siguiente.

    Step 6.

    Seleccionamos todos los perfiles, siguiente.

    Step 7.

    Continuación de MySQL

    Step 1.

    Para poder crear un usuario nuevo se usa el código señalado: create user ‘nombreDelUsuario’@’%’ identified by “contraseña”; Se usa el porcentaje para señalar una conexión remota.

    Step 2.

    Realizamos lo mismo pero ahora con localhost para que se reconozca el usuario localmente.

    Step 3.

    La acción anterior se puede comprobar revisando las columnas de la tabla user.

    Step 4.

    Con el comando mostrado le asignamos todos los privilegios al usuario mostrado, es decir puede acceder a toda la base de datos, esto señalado por el parámetro *.*.

    Step 5.

    Como podemos observar, ahora ingresamos con el nuevo usuario y nos permite ver las bases de datos, lo que significa que los permisos están bien asignados.

    Step 6.

    Cuando intentamos ingresar de manera remota nos podemos percatar que los permisos no están disponibles de esta forma, por lo que hay que asignarlos.

    Step 7.

    Ingresamos desde root y realizamos el cambio.

    Step 8.

    Cambio el cual podemos observar en la terminal remota.

    Step 9.

    A manera de práctica ingresamos un dato de forma remota en nuestra tabla de integrantes, que sería Pedro Alcocer con la matrícula E12345678, usaremos este registro para eliminar un dato.

    Step 10.

    Como se puede observar en la primera tabla, se encuentran tres datos, dos que previamente agregamos y el de Pedro el cual agregamos recientemente de forma remota, después aplicamos la linea de codigo: delete from integrantes where Nombre=”Pedro”; lo que nos permitió borrar esa fila de datos, lo cual se muestra en la tabla final.
